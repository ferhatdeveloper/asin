SumatraPDF (portable) — fiş sessiz yazdırma
============================================

RetailEX, kurulumda veya "npm run sumatra:fetch" ile buraya
SumatraPDF.exe koyarsanız fişi WebView yazdırma penceresi olmadan
varsayılan yazıcıya gönderir.

• Resmi site: https://www.sumatrapdfreader.org/
• Lisans: GPLv3 — kaynak: https://github.com/sumatrapdfreader/sumatrapdf
• Otomatik indirme: proje kökünde  npm run sumatra:fetch

Bu klasörde SumatraPDF.exe yoksa uygulama sistemde kurulu Sumatra veya
Program Files altındaki kurulumu kullanmaya devam eder.
